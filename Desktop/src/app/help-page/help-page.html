Navigation Manual

Hello and welcome to the webpage, as unusual as it seems the navigation is as easy as any other operating system.

Icons can be double clicked to open its contents, each window and icon can be individual draged on the desktop

Windows can be resized to fit you liking by pressing the Scale down and Scale Up icons attached to each window

Windows can be Minimized,Fullscrened,Exited

Taskbar functions as any other taskbar would, windows that are minimized will appear there as they are not exited and still active
