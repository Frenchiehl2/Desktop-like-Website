# Desktop-like-Website
A webpage that uses angular to recreate a fully functional operating system
# Build
install angular: <code>npm install @angular/cli</code>
 serve the website: <code>ng serve</code>
 
# About
As any other Desktop environment this features a fully custamizable desktop with icons and the bottim navbar
<p align="center"><img src="https://github.com/Frenchiehl2/Desktop-like-Website/blob/main/Docs/Desktop.png" width="600" height="300"></p>

Each Icon is linked to its respective window container that is created dynamicaly as per user request
<p align="center"><img src="https://github.com/Frenchiehl2/Desktop-like-Website/blob/main/Docs/Desktop-Window.png" width="600" height="300"></p>

At the bottom the nav bar is fully functional capable of saving idividual window states and returning to them once invoked again

<p align="center"><img src="https://github.com/Frenchiehl2/Desktop-like-Website/blob/main/Docs/Desktop%20Start%20menu.png" width="600" height="300"></p>

Each individual icon and window is fully dragable and each window is dynamicaly resizable just as in any other operating system
<p align="center"><img src="https://github.com/Frenchiehl2/Desktop-like-Website/blob/main/Docs/Desktop-dragables.png" width="600" height="300"></p>
